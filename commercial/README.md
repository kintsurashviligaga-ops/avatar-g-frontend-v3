# commercial/ — MyAvatar.ge 60s Showcase Commercial · render pipeline

Local assembly scaffold for the spec in [`../commercial_blueprint.md`](../commercial_blueprint.md).
This directory holds the **pipeline config + tooling + folder structure** — the heavy media
(source clips, audio, exports) are engine outputs and are git-ignored (see `.gitignore`).

## Layout
```
commercial/
├── asset_manifest.json      # machine-readable cut-list (S01–S16) + frame math + prompts + paths
├── AUTOMATION_MATRIX.md      # generated status matrix (regenerated by build_manifest.mjs)
├── pipeline/
│   ├── build_manifest.mjs    # parses the blueprint cut-list → validates frame math → scaffolds → emits manifest+matrix
│   └── render_pipeline.sh     # 7-stage assembly driver (guarded on asset presence); `--check` = dry-run
├── 01_source/{capture,ai_gen,mograph,hybrid}/   # S01–S16 clips, dropped by their engines
├── 02_audio/{vo,score,sfx,mix}/                  # VO1–VO4, score stems, sfx, mix_master.wav
├── 03_captions/                                  # kinetic Georgian caption alpha overlays
├── 04_work/                                      # intermediate renders (concat, capped picture)
├── 05_masters/                                   # ProRes 422 HQ picture master
└── 06_exports/{16x9_4k,9x16,1x1}/                # deliverables
```

## Workflow
1. **Generate / re-validate** the manifest + scaffold + matrix:
   ```bash
   node commercial/pipeline/build_manifest.mjs
   ```
   Fails loudly if the frame math drifts (contiguity, Σ = 1440, beats 240/480/480/240).
2. **Produce assets** with their engines (per the blueprint tags), drop into `01_source/**`,
   `02_audio/**`, `03_captions/`:
   - `[CAPTURE]` — real UI screen recordings (the biggest quality lever; §1.1).
   - `[AI-GEN]` / `[HYBRID]` — prompts + negatives live in `asset_manifest.json` per shot.
   - `[MOGRAPH]` — After Effects renders (Cap-3 node, ignite, logo, captions).
   - Audio — VO via ElevenLabs (cloned Georgian voice); score stems via MusicGen (§6, §7).
3. **Dry-run** — verify scaffold + which assets are present/pending (no render):
   ```bash
   bash commercial/pipeline/render_pipeline.sh --check
   ```
4. **Assemble** — each stage is guarded (refuses to run on a missing input):
   ```bash
   bash commercial/pipeline/render_pipeline.sh all   # stitch → mix → caption → master → encode → reframe
   ```

## Notes
- Master: **24fps · 3840×2160 · Rec.709 · 60.000s (1,440 frames)**. Audio **−14 LUFS / −1 dBTP**.
- Two intentional **hard cuts** (no xfade): S04 IGNITE (00:08:12) and S16 LOGO (00:55:00).
- `render_pipeline.sh stitch` produces an exact-60s **concat** skeleton; the premium **xfades +
  MoGraph transitions** are layered in the NLE/AE (xfade in ffmpeg needs transition handles on
  the source clips or the 60s runtime drifts — §10.1).
- FFmpeg command patterns for every stage are in the blueprint §10.
